# Changelog - AUY1104-CICLO-DE-VIDA-DEL-SOFTWARE-II

## [1.0.0] - 2026-04-23
### Added
- **Release Oficial:** Lanzamiento de la versión final estable para entrega académica.
- **Sincronización:** Merge completo de la rama `main` hacia `develop` para paridad de código.

- **Permisos Git:** Resolución de error 403 (Forbidden) mediante la configuración correcta de colaboradores y URLs remotas.
